# XP Quote API & Dashboard

Um sistema de monitoramento de ações desenvolvido com foco em performance e experiência do usuário (UX), seguindo a identidade visual da XP Investimentos.

##  Tecnologias Utilizadas

- **Backend:** C# .NET 8 (ASP.NET Core Web API)
- **Banco de Dados:** SQL (via Entity Framework Core)
- **Frontend:** HTML5, CSS3 e JavaScript (Dashboard responsivo)
- **Documentação:** Swagger UI

## Funcionalidades

- **Consulta de Ativos:** Integração para busca de cotações em tempo real.
- **Histórico SQL:** Persistência de consultas realizadas em banco de dados.
- **Interface XP:** Dashboard estilizado com as cores oficiais (Preto e Dourado).
- **Arquitetura Profissional:** Organização por pastas (Controllers, Data, wwwroot).

##  Estrutura do Projeto

```text
├── Controllers/         # Lógica da API e Rotas
├── Data/                # Configuração do SQL e AppDbContext
├── wwwroot/             # Frontend (HTML/JS/CSS)
├── Program.cs           # Configuração inicial do servidor
└── xp-quote-api-dashboard.csproj
